### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/GloChatv1/touch
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the server
   ```sh
   npm start
   ```
   or
   ```sh
   node server.js
   ```
6. Visit the site
   ```sh
   http://localhost:3000
   ```
   or
   ```sh
   http://192.168.x.x:3000
